    Postman Collection

    To facilitate the testing and understanding of the services we created a Collection of API calls that you can import in POST man an use it.

    Postman

    Postman is a powerful tool for performing integration testing with your API. You can find more information about Postman here: https://www.getpostman.com

    Instalation Steps

    1. Download Postman application and install it. https://www.getpostman.com/downloads

    2. Download the Collection file here

    3. Open Postman

    4. Go to File -> Import

    5. Find the location of the Collection file and press Open
